# Engram

Portable, local-first memory and context orchestration for AI tools.

Engram is being built for people who use more than one AI assistant and do not want to lose continuity when they switch tools. It is intended to provide a shared memory and knowledge layer that multiple AI clients can read from and write to through a single stable interface.

## Status

Engram is pre-release. v0 is in design. No installable artifacts yet.

This repository will grow as v0 implementation begins. User-facing documentation (quickstart, self-host guide, API reference) will be added when there is something to install, run, and call.
