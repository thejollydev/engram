# Public Roadmap

## Product Themes

### Phase 1

- Canonical memory model
- Multi-plane retrieval architecture
- Local-first API and MCP surface
- Filesystem and Obsidian ingestion
- Early UI for memory inspection and editing

### Phase 2

- Provider/plugin interfaces
- Early graph support
- Early document-provider support with a native fallback and PageIndex integration
- Model-memory importers
- Better lifecycle and temporal reasoning

### Phase 3

- Knowledge packs
- Context-pack tuning and explainability
- Additional connectors
- Hosted deployment options and managed services

### Future

- Team and org scopes
- governance and policy controls
- managed connectors
- hosted control plane
- commercial support offerings
